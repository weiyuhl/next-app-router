## 🎯 URL 路径映射

### 前台路由（使用前台布局）
```
/                → src/app/(frontend)/page.tsx
/about           → src/app/(frontend)/about/page.tsx
/qingyu          → src/app/(frontend)/qingyu/page.tsx
/code-snap       → src/app/(frontend)/code-snap/page.tsx
```

### 后台路由（使用后台布局）
```
/dashboard       → src/app/(admin)/dashboard/page.tsx
/test-routes     → src/app/(admin)/test-routes/page.tsx
```

### 认证路由（使用认证布局）
```
/login           → src/app/(auth)/login/page.tsx
/register        → src/app/(auth)/register/page.tsx
/forgot-password → src/app/(auth)/forgot-password/page.tsx
```

**注意**：路由组名称 `(frontend)`、`(admin)`、`(auth)` 不会出现在 URL 中！
